# 🎯 Number Guessing Game

This is a Python-based number guessing game where the user tries to guess a randomly generated number within limited attempts.

## 🚀 Features
- Random number generation
- Limited attempts (6 tries)
- Smart hints (Very close, Near, Far)
- Direction guidance (higher/lower)
- Input validation

## 🧠 How It Works
The program generates a random number between 1 and 50.  
The user tries to guess it within 6 attempts.  
After each guess, the program provides feedback based on how close the guess is.

## 💡 What I Learned
- Using loops for repeated attempts
- Conditional logic for decision making
- Handling user input safely
- Building interactive command-line applications
